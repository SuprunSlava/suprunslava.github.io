Что такое AJAX? Как принципиально устроена эта технология?

AJAX - asynchronous javascript and xml. Суть этой технологии - асинхронная работа веб приложения, к примеру, выполнение запросов в фоне. Эта технология позволяет асинхронно взаимодействовать с элементами страницы при этом не вызывая перезагрузку самой страницы. Т.е. к примеру чаты, вылезающие на некоторых веб-страницах, работают по технологии AJAX, любые сообщения, которые были отправлены или получены, без AJAX вызывали бы перезагрузку страницы с последующим перерендером DOM. При обычной модели запроса, со страницы отправлялся бы HTTP запрос, сервер получает запрос, отсылает ответ, браузер получает ответ и перезагружает страницу. С AJAX сперва создается вызов JS, который активирует фетч данных (в старых версиях xmlhttprequest), в фоновом режиме происходит запрос данных и при получении обновляется только определенные элементы DOM структуры.
