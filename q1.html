Чем отличаются Web server и Application server?

Web server работает с такими протоколами как http и https. К примеру можно привести Apache httpd - он является веб сервером и для веб сервера мы используем JSP или Сервлеты. В случае application server'а - он работает с любым протоколом, к примеру JBoss или Tomcat (является и веб сервером и сервером приложений одновременно, т.к. имеет контейнер сервлетов обрабатывающий запросы). На сервере приложения мы можем разместить EJB, Web-server, WebSphere или другую бизнес логику.
